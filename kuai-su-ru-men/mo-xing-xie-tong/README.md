# 模型协同

这里以用户A、B、C对仓库“Library”中的模型“Control”协同操作为例，其中A为仓库创建者，B和C为协作者。

模型“Control”由子模型“Ramp”和“Gain”组成，其中子模型“Ramp”由用户B负责搭建，子模型“Gain”由用户C负责搭建，最后由用户A进行组合。
